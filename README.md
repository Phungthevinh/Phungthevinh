### 🛡️ Featured Projects

#### 1. [zero_trust_gateway](https://github.com/Phungthevinh/zero_trust_gateway) (Rust)
> An AI-Native Zero-Trust API Gateway and high-performance Semantic Cache.
* **Core Engine**: Developed on top of `axum` and `tokio` for lightning-fast performance.
* **Security & Zero-Trust**: Integrated sub-1.2ms request blocking, JWT authorization, and cryptographic signature validation using Ed25519 (`ring`).
* **AI-Native Cache**: Built local ONNX-based embedding search (`tract-onnx`) to construct a Semantic Cache in memory, significantly lowering LLM invocation latency and costs.
* **MCP Integration**: Parsed and handled Model Context Protocol (MCP) flow directly at the gateway layer.

#### 2. [webAPI_Personnel](https://github.com/Phungthevinh/webAPI_Personnel) (C#)
> Web API for managing corporate personnel and Koi fish farming assets.
* Built using ASP.NET Core, focusing on Clean Architecture, RESTful API design, and structured databases.

#### 3. [HELPDESK.AI](https://github.com/Phungthevinh/HELPDESK.AI) (JavaScript / AI)
